This is a minimal starter skeleton for PromptForge (local web app to craft AI prompts).
Run the backend:
```bash
cd backend
python -m pip install -r requirements.txt
python app.py
```
Open http://127.0.0.1:11435 in your browser.

Config file: `~/.promptforge/config.yaml` (optional). Default Ollama URL: http://localhost:11434

This starter is intentionally small; use it as a base for the full spec.
